Issue Guide

Update failed

Use this guide if a BOOX firmware update stops, reports an error, or leaves the device unstable afterward.

Firmware Install error Recovery

Common signs

  • The update stops before finishing.
  • An error message appears during installation.
  • The device restarts but behaves abnormally after the update.
  • The issue started during a low-battery or unstable Wi-Fi session.

Possible causes

  • The battery charge was too low for the update to complete safely.
  • The download was incomplete or corrupted.
  • Storage space or a background process interrupted installation.
  • The device may need model-specific recovery help.

Step-by-step troubleshooting

  1. Charge the device first. Updates should not be retried on a low battery.
  2. Restart once if the device still responds. A clean reboot can clear temporary installation states.
  3. Check your Wi-Fi connection. If the update downloads again, use a stable network before retrying.
  4. Free up storage if the system is accessible. Very low free space can interfere with updates.
  5. Do not keep retrying back-to-back. Repeated failed attempts can make the device harder to recover.
  6. Move to official support if startup or system stability is affected. Recovery steps can vary by model and firmware version.

Important warning

A factory reset may not fix a failed firmware install and can add data loss to an already unstable system. Use it only if BOOX support recommends it.

When to contact support

Contact BOOX support if the update fails more than once, the device becomes stuck during startup, or core functions stop working after the install attempt.
