Issue Guide

Wi-Fi not connecting

Use this guide if the device sees your network but will not join, keeps failing authentication, or disconnects immediately.

Wi-Fi Network Internet

Common signs

  • The network name appears but connection fails.
  • The password is accepted on other devices but not on BOOX.
  • The connection drops right after joining.
  • Only one specific Wi-Fi network fails.

Possible causes

  • The saved network settings are incorrect or outdated.
  • The router is using a band or security mode the device does not like.
  • The network signal is weak or unstable.
  • The issue may be temporary router-side congestion.

Step-by-step troubleshooting

  1. Forget the network and reconnect. Re-enter the password carefully to clear a bad saved profile.
  2. Move closer to the router. Weak signal can look like an authentication or stability problem.
  3. Restart both the BOOX and the router. This clears temporary connection states on both sides.
  4. Test another Wi-Fi network if possible. That helps show whether the issue is with the device or one router.
  5. Check whether the router is using unusual settings. If you control the router, simpler security settings and a standard band can improve compatibility.
  6. Leave resets as a last resort. Most Wi-Fi problems are solved by reconnecting, restarting, or adjusting router settings.

Important warning

Do not factory reset the BOOX just because one Wi-Fi network fails. Test another network first so you do not erase data for a router-side issue.

When to contact support

Contact BOOX support if the device cannot connect to any network after restart and reconnection, especially if Wi-Fi used to work normally on the same hardware.
