Screen ghosting

If faint remnants of previous pages stay visible, the display may need a refresh setting adjustment or a full redraw.

Display refresh E Ink behavior Visual artifacts

Common signs

  • Faint text or images remain visible after the page changes.
  • Ghosting becomes more obvious after scrolling or rapid page turns.
  • A full refresh clears the issue temporarily.
  • The screen content is readable, but looks smudged or layered.

Possible causes

  • The current refresh mode is optimized for speed instead of image cleanup.
  • The app you are using is refreshing too infrequently between page changes.
  • The display needs a manual full refresh after extended use.
  • System settings or app settings are reducing display quality to improve responsiveness.

Step-by-step troubleshooting

  1. Run a full screen refresh. Use the device refresh gesture, navigation ball, or refresh option to clear leftover images.
  2. Switch to a different refresh mode. Try a clearer mode such as Normal or Regal if a fast mode is causing extra ghosting.
  3. Check the app's display settings. Some reading or note apps let you choose refresh frequency or visual optimization.
  4. Reduce rapid scrolling. E Ink displays can show more artifacts when content moves quickly for long periods.
  5. Restart the device. A restart can clear temporary display behavior issues and reset stuck rendering states.
  6. Test on another screen or app. If ghosting only happens in one app, the problem may be app-specific rather than hardware-related.

Important note

Some light ghosting is normal on E Ink screens, especially in faster refresh modes. It becomes a problem when a full refresh no longer clears it well.

When to contact support

Contact BOOX support if ghosting remains severe after full refreshes, appears across every app and screen, or is combined with flickering, dead areas, or touch problems.
